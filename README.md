# Mirth Open Blazor

Repositório que contém o projeto **LabIntegrator**, uma solução .NET para simular integrações laboratoriais com:

- **API REST** (`LabIntegrator.Api`) para canais e mensagens
- **Serviço legado** (`LabIntegrator.LegacyService`) que lê arquivos Mindray e envia para a API
- **Dashboard Blazor** (`LabIntegrator.Dashboard`) para visualizar mensagens em tempo real
- **Biblioteca de domínio** (`LabIntegrator.Core`)
- **Infraestrutura EF Core** (`LabIntegrator.Infrastructure`)
- **Testes automatizados** (`LabIntegrator.Tests`)

O README detalhado com instruções de setup, scripts e execução está em `LabIntegrator/README.md`.
